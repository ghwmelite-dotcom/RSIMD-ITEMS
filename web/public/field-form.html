System Data Entry Guide

After completing fieldwork, enter findings into RSIMD-ITEMS:

1. Open rsimd-items.pages.dev or tap the app icon on your phone
2. Scan QR on equipment (or tap Maintenance → Log Activity)
3. Select Location (directorate/unit) and Room Number from this form
4. Choose Maintenance Type (RTN = Routine, COR = Corrective, EMR = Emergency)
5. Select Category matching the "Issue Category" column from this form
6. Copy the Description and Resolution from your notes (or use voice input 🎤)
7. Attach photos if taken. Submit. Works offline — logs sync when back online.

Tip: Create one log entry per issue category per room. Multiple issues in the same room = multiple logs.
